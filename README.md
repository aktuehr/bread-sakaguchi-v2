# bread-sakaguchi-v2

パン工房 SAKAGUCHI のホームページ。

## Directories

### public

静的リソース(画像)を配置

## Pages

### /

トップページ

### /about

お店についてのページ

### /category

パンの紹介のページ

### /access

アクセスのページ

### /feature/\*

企画やお知らせなどのページ

## Getting Started

以下のコマンドで開発状態のものを確認できます。

```bash
npm run dev
```

## Export

以下のコマンドで静的リソースとして export できます。

```bash
npm run build
```
